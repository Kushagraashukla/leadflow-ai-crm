# MVP Scope Definition

## Included in MVP

### Authentication

* User Registration
* User Login
* User Logout

### Lead Management

* Create Lead
* View Leads
* Update Lead
* Delete Lead

### Notes Management

* Add Notes to Leads
* View Notes History

### Sales Pipeline

* New
* Contacted
* Qualified
* Proposal Sent
* Won
* Lost

### Dashboard

* Total Leads
* Leads by Status
* Recent Activity

### AI Features

* AI Generated Lead Summary

---

## Excluded from MVP

* Multi-Organization Support
* Role-Based Permissions
* Email Integration
* Calendar Integration
* Stripe Billing
* Advanced Analytics
* Kanban Drag-and-Drop
* Mobile Application

---

## Reasoning

The goal of the MVP is to deliver a working CRM application with core business functionality while maintaining a manageable scope for rapid AI-assisted development.

Features have been prioritized based on business value, implementation complexity, and assessment requirements.
