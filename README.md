## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or pnpm
- Git 2.x

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/leadflow-ai-crm.git

# Navigate to project directory
cd leadflow-ai-crm

# Install dependencies
npm install

# Copy environment variables
cp .env.example .env

# Start development server
npm run dev
```
